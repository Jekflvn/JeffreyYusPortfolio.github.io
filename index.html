Pixelman Platformer

Pixelman Platformer is a platforming game I developed as practice before creating my dream game. In Pixel Platformer, run and jump your way through multiple, well decorated environments full of enemies and reach the goal. I designed everything about the game besides the assets, which were taken from the unity store. This meant creating all the game objects and corresponding scripts for everything. The game features all mechanics to be expected of a platformer such as fluid player movement, collectables, death logic, animations, and a level selector. I still plan to add more environments, mechanics, and levels to the game once I finish working on other projects, but you can find the game on my github and try it out.

High-Speed Concurrent Server

I launched a server to effectively handle concurrent client requests to modify server-side states. It simulates how a client controls a sensor or actuator and how the entity sends data back to a client. While it is simply a simulation for now, the design could theoretically be implemented to work on object in the real world. The design employs multi-threaded design to support a high number of requests per second and Ensured synchronization safety and utilized thread-safety principles to avoid deadlocks and race conditions.
